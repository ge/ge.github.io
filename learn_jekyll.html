learning_jekyll: Looping over posts in site.categories

The variable site.categories.CATEGORY is defined in the jekyll documentation as the list of all Posts in category CATEGORY.

If this is correct and if I have a category travel I should be able to list the posts in this category by referring to site.categories.travel. But this does not work for me.

However, I can still work with site.categories. I now believe that site.categories is a list of objects containing two methods:

  • post[0] one of the post's categories
  • post[1] the content of the post

This loop prints out the category for each post in site.categories: 

<div class="posts">
    <ul>  
    {% for post in site.categories %}
      <li>
          <span>
              Post category: [{{post[0]}}]
          </span>
      </li>  
    {% endfor %}
    </ul>
</div>
